# Euclidean norm of a sparse_numeric vector

Computes the Euclidean (L2) norm \$\$\sqrt{\sum_i x_i^2}\$\$.

## Usage

``` r
norm(x, ...)

# S4 method for class 'sparse_numeric'
norm(x, ...)
```

## Arguments

- x:

  A `sparse_numeric` object.

- ...:

  Ignored; included for compatibility with the generic.

## Value

Numeric scalar norm.
